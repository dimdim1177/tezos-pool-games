◆ manager

address CPoolGames::t_pool::manager

Pool manager (admin of this pool only)

If the pool-as-a-service is enabled in the code with the ENABLE_POOL_AS_SERVICE key, then it is the only pool manager, the owner and the contract administrator do not have access to the pool. When the pool-as-a-service is turned off, it can be used to increase the security of the backend. The contract must be called by an external backend for the operation of the drawings, you can use the manager's access for this, and the owner's keys are not stored on the backend server

See also
ENABLE_POOL_AS_SERVICE
