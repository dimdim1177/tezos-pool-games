typedef nat MRandom::t_random

Random number.

To create a random number, the hash of the nearest Tezos is taken, the time of which is greater than or equal to the time of the event, SHA-256 is removed from it, we get the levelHash value. Also, the SHA-256 hash is removed from the line "customer's address<space><object ID in the customer's contract in decimal notation>", we get the requestHash value. A random number is calculated as random = levelHash^ requestHash and converted to nat format. As a result, in general, we have a 256-bit random number
