Detailed Description

Contract admins management module.

Adding a new admin by any admin, removing any admin by any admin, including himself. The last admin cannot delete himself so that at least one admin remains in the contract. Example using a module without other access modules

#Include "module/MAdmins.ligo"
type t_storage record [
admins: MAdmins.t_admins;
...
];
type t_entrypoint is
| AddAdmin of MAdmins.t_admin
| RemAdmin of MAdmins.t_admin
...
function main(const entrypoint: t_entrypoint; var s: t_storage): t_return is
case entrypoint of
| AddAdmin(params) -> (cNO_OPERATIONS, { s.admins := MAdmins.accessAdd(params, s.admins); } with s)
| RemAdmin(params) -> (cNO_OPERATIONS, { s.admins := MAdmins.accessRem(params, s.admins); } with s)
...
