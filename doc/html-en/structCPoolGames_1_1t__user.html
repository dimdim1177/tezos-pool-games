◆ addWeight

t_weight CPoolGames::t_user::addWeight

Additional user weight only for AlgoTimeVol at (tsBalance >= game.tsBeg) && (tsBalance < game.tsEnd)

When depositing/debiting during the game, this may change the user's weight in the draw at AlgoTimeVol This variable stores the user's accumulated weight from the beginning of the game.tsBeg game to tsBalance and in the following games (with tsBalance < game.tsBeg) the variable is ignored so as not to make unnecessary data updates
