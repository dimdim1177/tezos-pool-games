#define ENABLE_TRANSFER_SECURITY   false

As far as possible, the most secure transfer of tokens.

If enabled, when transferring tokens, approve AMOUNT|add_operator is done first, after transfer, approve 0|remove_operator. If disabled, approve MAXAMOUNT|add_operator is enabled once and is no longer revoked
