◆ winWeight

t_weight CPoolGames::t_game::winWeight

Вес для определения победителя

Вес определяется делением с остатком 256-битного случайного числа на вес партии + 1n (то есть никогда не равен 0) Для поиска победителя бэкенд должен взять адреса всех участников пула, отсортировать их в алфавитном порядке и последовательно суммируя веса участников найти того, при добавлении веса которого результат >=winWeight И <=winWeight, он и будет являться победителям розыгрыша Участники, вошедшие в пул с нарушением ограничений (например, менее чем за minSeconds до окончания партии), будут иметь для текущей партии вес 0 и таким образом исключаются из победителей.
