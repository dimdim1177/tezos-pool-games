◆ manager

address CPoolGames::t_pool::manager

Менеджер пула (админ только данного пула)

Если в коде включен пул-как-сервис ключом ENABLE_POOL_AS_SERVICE, то он единственный управляющий пулом, владелец и админ контракта не имеют доступа к пулу. При выключенном пул-как-сервис может использоваться для увеличения безопасности бэкенда. Контракт должен вызываться внешним бэкендом для работы розыгрышей, можно использовать для этого доступ менеджера, а ключи владельца не хранить на сервере бэкенда

См. также
ENABLE_POOL_AS_SERVICE

См. определение в файле CPoolGames/types.ligo строка 414
