Адреса, которым уже одобрили использование токенов контракта

Для уменьшения кол-ва операций, одобряем ферме использование токенов контракта только один раз Одобрения идентифицируются по комбинации адреса и токена, поэтому для нескольких ферм с одинаковыми адресами и токенами, но разными ID будет только одно одобрение на всех

◆ usedFarms

t_farms CPoolGames::t_storage::usedFarms

Использованные фермы

Если два пула будут одновременно использовать одну ферму это приведет к тому, что одна заберет вознаграждение другой, потому что списывается вознаграждение за всех пользователей, которые могут вносить и извлекать депозиты в любой момент времени. Поэтому фиксируем использованные фермы и запрещаем создавать пулы с уже существующими фермами
