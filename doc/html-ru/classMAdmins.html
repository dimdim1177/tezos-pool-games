Подробное описание

Модуль управления админами контракта

Добавление нового админа любым админом, удаление любым админом любого админа, включая самого себя. Последний админ не может удалить сам себя, чтобы в контракте остался хотя бы один админ. Пример использование модуля без других модулей доступа

#Include "module/MAdmins.ligo"
type t_storage record [
admins: MAdmins.t_admins;
...
];
type t_entrypoint is
| AddAdmin of MAdmins.t_admin
| RemAdmin of MAdmins.t_admin
...
function main(const entrypoint: t_entrypoint; var s: t_storage): t_return is
case entrypoint of
| AddAdmin(params) -> (cNO_OPERATIONS, { s.admins := MAdmins.accessAdd(params, s.admins); } with s)
| RemAdmin(params) -> (cNO_OPERATIONS, { s.admins := MAdmins.accessRem(params, s.admins); } with s)
...

Удаление админа безусловно

Проверка прав на удаление админа должна делаться извне Нельзя удалить админа, когда остался только один и нет владельца контракта, который может добавить админа, будет возвращена ошибка cERR_REM_LAST_ADMIN Если админ для удаления не найден, будет возвращена ошибка cERR_NOT_FOUND
