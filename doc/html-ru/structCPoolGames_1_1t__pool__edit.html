Подробное описание

Данные для редактирования пула

Все настройки, необходимые для редактирования пула. Если любое из полей None, оно игнорируется в методе редактирования, редактировать можно только остановленный пул у которого закончился розыгрыш

См. также
EditPool, t_pool, PoolStatePause, GameStatePause
